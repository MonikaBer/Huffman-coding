# Huffman-coding
Coding and decoding text files by Huffman code in MIPS32 Assembly.

1) Text to coding should put in file named 'file1'. 
2) After launching program and choosing an option '0' (coding), coded text will put in file named 'file2'. 
3) After next launching program and choosing an option '1' (decoding), decoded text will put in file named 'file3'.
